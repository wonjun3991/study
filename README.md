# Study Vault

개인 학습 노트 Obsidian Vault.

## Structure

```
.
├── inbox/          # 새 노트 기본 저장 위치
├── daily/          # Daily notes
├── templates/      # 노트 템플릿
├── attachments/    # 이미지, 파일 첨부
└── .obsidian/      # Obsidian 설정
```

## Usage

1. Obsidian에서 이 디렉토리를 vault로 열기
2. `inbox/`에 새 노트가 자동 생성됨
3. 정리 후 적절한 폴더로 이동
